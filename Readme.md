## 1. Markdown 基本語法
### 1-1. 標題
* 第一種方式:
    * 在標題文字後加上`===`或`---`來表示主副標題
    ```Markdown
    主標題
    ======
    副標題
    ------
    ```
* 效果如下:

    主標題
    ======
    副標題
    ------

* 第二種方式:
    * 使用`#`來表示標題與標題的級別
    ```Markdown
    # Heading Level 1(字體最大)
    ## Heading Level 2(字體次大)
    ### Heading Level 3(字體適中)
    #### Heading Level 4(字體較小)
    ##### Heading Level 5(字體次小)
    ###### Heading Level 6(字體最小)
    ```
* 效果如下:

    # Heading Level 1(字體最大)
    ## Heading Level 2(字體次大)
    ### Heading Level 3(字體適中)
    #### Heading Level 4(字體較小)
    ##### Heading Level 5(字體次小)
    ###### Heading Level 6(字體最小)

### 1-2. 字體效果
* *斜體字* or _斜體字2_
    ```Markdown
    *斜體字* or _斜體字2_
    ```
* **粗體字**
    ```Markdown
    **粗體字**
    ```
* *斜體 and 粗體字* or __斜體 and 粗體字2__
    ```Markdown
    *斜體 and 粗體字* or __斜體 and 粗體字2__
    ```
* ~~刪除線~~
    ```Markdown
    ~~刪除線~~
    ```
* 文字^上標^ or 文字~下標~
    ```Markdown
    文字^上標^ or 文字~下標~
    ```
* ++底線++
    ```Markdown
    ++底線++
    ```
* ==螢光標記==
    ```Markdown 
    ==螢光標記==
    ```
